  • Creating sequence nos.

    1. SubString(AsString(GetEnv("CurRecordNumber") + 100000), 1) ex: 00123
    2. AsString(GetEnv("CurRecordNumber") + 10000) ex: 10123
    3. "2" & FormatNumber(GetEnv("CurRecordNumber"), "0000") ex: 20123
    4. SubString("##########", 0, 6 - Length(GetEnv("CurRecordNumber"))) & GetEnv("CurRecordNumber") ex: ###123
    5. GetEnv("CurRecordNumber") & SubString("**********", 0, 6 - Length(GetEnv("CurRecordNumber"))) ex: 123***

  • QLingo string functions

    • Lcase("HELLO") = "hello"
    • Tcase("hello world") = "Hello World"
    • UCase("hello") = "HELLO"
    • Length("abc") = 3
    • IsNullOrEmpty(|->[field]) = true or false
    • SubString("hello",1,2) = "el"
    • Trim(" Hello World ") = "Hello World"
    • LTrim(" Hello World ") = "Hello World "
    • RTrim(" Hello World ") = " Hello World"
    • Find("Hello World", "o", 5) = 7 // returns -1 if value not found
    • Replace("Hello World", "old w", 6, 1) = "Hello old world"
    • FindAndReplace("Hello World", "World", "Earth") = "Hello Earth"
    • FindAndReplaceChars("A1B2C3D4E5", "12345", "*") = "A*B*C*D*E*"
    • CleanNumber("A1B2!3@4?5") = "12345"
    • "Hello" + HexToUnicode("0020") + "World" = "Hello World" // 0020 = 'space'; 0009 = 'tab'

  • Steps to re-create local SQLDB

    1. Run CMD.
    2. Type "SQLLocalDB info" to confirm that MSSQLLocalDB is installed.
    3. SQLLocalDB stop MSSQLLocalDB
    4. SQLLocalDB delete MSSQLLocalDB
    5. Go to "C:\users\[name]" folder ("%UserProfile%") and delete all files with .ldf and .mdf extensions, if they exist.
    6. SQLLocalDB create MSSQLLocalDB
    7. SQLLocalDB start MSSQLLocalDB
